# ☕ Java + Spring Boot Mastery

Complete Java learning repository — from Core Java basics
to Spring Boot REST APIs, Servlets, Hibernate, and Docker.

## 📁 What's Inside

| Folder | Topics Covered |
|---|---|
| `java_intro` | Java basics, data types, control flow |
| `java-oops` | Classes, inheritance, polymorphism, interfaces |
| `java-jdbc` | Database connectivity, CRUD with raw JDBC |
| `java-servlet` | Servlet lifecycle, HttpServlet, doGet/doPost |
| `java-hibernate` | ORM, Entity mapping, HQL queries |
| `java-rest-api` | RESTful API design, HTTP methods, status codes |
| `java-springboot` | Spring Boot 3, @RestController, JPA, MySQL |
| `java-authentication` | JWT, Spring Security, BCrypt |
| `java-docker` | Dockerfile, docker-compose, containerization |
| `mini-project` | End-to-end mini projects |

## 🛠️ Tech Stack
Java 21 · Spring Boot 3 · Hibernate · JDBC · 
Servlets · REST API · Docker · MySQL · Maven
